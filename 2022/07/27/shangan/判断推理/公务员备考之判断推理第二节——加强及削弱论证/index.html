公务员备考之判断推理第二节——加强及削弱论证

2022-07-27 22:47:32
  • 判断推理
  • 考公
  • | 判断推理

一、论证概述

1、题型识别

  • 提问方式：最能 支持/削弱 论点的是

2、知识点

2.1 论证结构及原则

论证则是通过论据证明论点

只有论点是必须存在的，所以 优先找论点

论证结构

论据 => 论证过程 => 论点

  • 论据：得到论点的根据/依据 (可有可无)
  • 论证过程：论据和论点之间的(隐含)联系 (可有可无)
  • 论点：被证明的结论/观点 (一定有)

原则：话题保持一致

2.2 解题思路

根据论证题目的原则， 话题不一致的选项可以直接排除

对比择优

  1. 找论点：
    • 引导词：认为/以为/观点是/表明/说明/结果是/因此/所以/建议
    • 首尾句：题目一般为总分或分总结构
    • 主旨概括：少量题目需要自己概括论点
  2. 找论据
    • 证明论点的根据/依据；举例子等
  3. 看要求
    • 加强 = 赞同
    • 削弱 = 反对
  4. 选答案
    • 对比择优，按照强度排序
    • 与题目的关联性 (一般关联越密切强度越强；若都无关，尊重客观事实)
    • 方法强度排序

二、加强论证

1、题型识别

  • 提问方式：加强、支持、肯定、前提、假设

2、知识点

2.1 加强论据 (🌟)

排除他因

加强论点为真的可能性

排除其他选项，加强唯一原因的可能性

补充论据
  • 讲道理：说明原因/原理
  • 增加新证据、事实、例子、数据、实验等
  • 和论点方向一致
对比实验

需注意控制变量一致等

有A有B，无A无B

2.2 建立联系【搭桥】(🌟🌟)

建立论据和论点之间的联系 将隐藏联系明确化

  • 题干论据论点关系不紧密(话题不完全一致)
  • 正确选项同时具备论据和论点的关键信息
  • 前提、假设文法中考察搭桥比重偏高

2.3 加强论点(🌟)

加强论点即为重复论点

2.4 不能加强

提问方式：不能加强/不能支持/不能肯定

⚠️注意：

不能加强 不等于 削弱；

无关项、削弱项均不能加强 (如果选项中二者都有，优先选削弱)

三、削弱论证

1、题型识别

  • 提问方式：质疑、削弱、否定、反驳、反对

2、知识点

2.1 否论据

另有他因(🌟)

他因和无关项的区别： 是否和论点有联系，有联系的为他因

引入新的可能性，削弱了本身原因的可能性

反向论据(🌟)

与论点相反的证据、道理、事实、例子、数据、实验等

对比实验(🌟🌟)

打断A、B同时存在、同时增强的关联

有A有B，有A无B；

有A有B，无A有B；

2.2 打断联系【拆桥】(🌟🌟)

打断据和论点之间的联系

2.3 否论点

直接否论点(🌟🌟🌟)
  • 题目中只有论点，可直接否定论点
  • 题目中有论据，需要有理由的否论点
否前提(🌟🌟🌟)

否定论点成立必不可少的条件。例如：小明活着，否前提则可以为：小明处于无氧环境中

因果倒置(🌟🌟🌟)

论点：A导致B (因为A，所以B)

倒置：B导致A (因为B，所以A)

2.4 不能削弱

提问方式：不能削弱/不能质疑/不能否定/不能反驳

⚠️注意：

不能削弱 不等于 加强；

无关项、加强项均不能削弱 (如果选项中二者都有，优先选加强 )
