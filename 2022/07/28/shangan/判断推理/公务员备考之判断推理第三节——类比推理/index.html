公务员备考之判断推理第三节——类比推理

2022-07-28 01:08:44
  • 判断推理
  • 考公
  • | 判断推理

一、基础概念

类比推理是给一组相关词，通过观察分析，找出一组与之在逻辑关系上最为贴近或相似的词

1、解题原则

  • 二元关系：逻辑关系
  • 二级辨析：找到最贴近/相似的词

2、题型分类

  • 两元型：两个词(X : Y)
  • 多元型：一般为三个词(X : Y : Z)
  • 填空型：( ) 对于 X 相当于 Y 对于( )

填空型需要逐一代入，验证前后关系

二、外延关系

1、全同关系

两个词概念意思完全相同

  • 土豆 ：马铃薯
  • 芙蕖(菡萏) ：荷花
  • 杜鹃花 ：映山红
  • 春夏秋冬 ：四季

2、交叉关系

两个词有部分相交，但不能完全相交

推理方法：

  • 分类标准不同
  • 造句(下面4句都需符合)
    • 有的 A 是 B，有的 A 不是 B
    • 有的 B 是 A，有的 B 不是 A
  • 例如：水桶 ：木桶
    • 水桶是功能分类；木桶是材料分类
    • 四句造句都通顺合理

3、并列关系

相同的分类标准下对概念分类，可分为矛盾关系和反对关系

3.1 矛盾关系

矛盾关系指概念分类后，有且只有两个子集

例如：生：死； 南：女； 转基因：非转基因

3.2 反对关系

反对关系指分类后，还有其他子集

例如：苹果：香蕉；黑：白；鼠标：键盘

4、包含关系

一个完全包含另外一个，需注意前后顺序，可分为种属关系和组成关系

4.1 种属关系

种属关系为类与类的对应，可造句为：xx是xx

例如：树：杨树 (杨树是树)

4.2 组成关系

组成关系是整体和部分的对应，可造句为：xx是xx的一部分

例如：树：树根(树根是树的一部分)

三、语义关系

  1. 语义关系靠积累
  2. 利用会的去排除
  3. 实在不会可以猜

1、近义&反义关系

两个词互为近义或反义词关系

例如：知难而退：百折不挠(反义)；

二级辨析(褒/贬+前后顺序)
  • 近义关系程度有区别：例如尊重：崇敬(程度强)；承受：忍受(程度强)
  • 语义关系情感色彩区别 (褒义/贬义)

2、象征关系

象征关系分为传统象征和比喻象征。

  • 传统象征
    • 巾帼：女子； 须眉：男子
    • 没话：坚强； 桑梓：家乡
  • 比喻象征
    • 心灵的窗户：眼睛
    • 踢皮球：推诿
    • 纸老虎：外强中干
    • 白鸽：和平

四、内涵关系

1、条件关系

条件关系主要观察词语之间的必然性

  • 充分条件：有了就一定行
    • 例如：燃烧：放热
  • 必要条件：没有一定不行
    • 例如：18周岁：选举权

2、因果关系

因果关系主要考察事件之间的因果联系，大部分因果关系是非必然的

因果关系为必然的称之为条件关系

二级辨析
  • 结果：结果为消极还是积极
    • 竞争：淘汰(消极)
    • 创业：成功(积极)
  • 原因：原因是人造还是自然(‼️)
    • 玻璃幕墙(人造)：光污染
    • 火山喷发(自然)：地震

3、顺承关系

顺承关系需重点观察时间上的先后顺序

需尊重普遍逻辑和常规规律

二级辨析
  • 主体是否相同
    • 恋爱：结婚(主体一致)
    • 请假：批准(主体不一致)

4、属性关系

属性关系指概念具备的性质。分为必然属性和或然属性

  • 必然属性：醋：酸
  • 或然属性：花：红

5、对应关系

对应关系主要看搭配关系，重点看四种搭配

  • 与“物”相关：

    • 物[物品/人物/事件]：场所
      • 火车：铁轨
      • 病人：医院
      • 炒菜：厨房
    • 物：功能（二级辨析：功能为主要还是次要）
      • 电灯：照明(主要功能)
      • 钢笔：书写(主要功能)
      • 白醋：消毒(次要功能)
    • 物：工艺（二级辨析：为物理还是化学变化）
      • 陶瓷：煅烧(化学变化)
      • 白酒：发酵(化学变化)
      • 玉器：磨制(物理变化)
    • 物：材料
      • 书桌：木材
      • 面粉：小麦
      • 斗笠：竹子

  • 与“职业”相关

    有一些常见的假职业：

    学生、病人、志愿者、网民、经理…

    • 职业：工作
      • 护士：输液
      • 清洁工：保洁
    • 职业：工具
      • 医生：听诊器
      • 裁缝：剪刀
    • 职业：对象
      • 医生：病人
      • 教师：学生
    • 职业：产出
      • 厨师：菜品
      • 摄影师：照片

  • 与“逻辑”相关

    • 方式：目的
      • 凿壁：偷光
      • 种树：绿化

  • 与”常识“相关

    • 成语：主人公：出处
      • 负荆请罪：廉颇/蔺相如：史记
    • 诗词：作者：名称：意象
    • 其他常识
      • 周：东周：西周

6、语法关系

  • 主谓宾关系
    • 主语：动作的发起者
    • 谓语：谓语动词
    • 宾语：动作的承受者
    • 例如：记者：采访(主谓关系)；记者：新闻(主宾关系)
  • 偏正关系(修饰词+中心语)
    • 形容词(的) + 名词
      • 漂亮：女孩
    • 副词(地) + 动词
      • 认真：学习
