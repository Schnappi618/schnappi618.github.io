公务员备考之判断推理第四节——图推位置及样式规律

一、题目形式

图形推理主要有以下几种题目形式

  • 一排型

    • 一般为 5+? (少量4+?) 模式
    • 特征：从左到右遵从稳定规律

  • 两组型

    • 特征：左边找规律，右边用规律(规律不一定完全相同)

  • 九宫格

    • 特征：一般横行分组，少量题目存在竖行、斜行、米字型、O型、S型等

    • 一般第一组找规律；第二组验证规律；第三组使用规律

  • 分组型

    • 特征：6个已知图形，分成3对3两组

二、位置规律

1、题型识别

当图形组成相同，优先考虑位置规律

例如上图例子：每个图中都有16个球，包含两个黑球，其余为白球；只有4图中为一个黑球，考虑位置调整过程中，两个黑球重合的情况

则认为图形组成相同

2、动态规律

动态规律主要指：位置形态是不断变化的。

主要包含平移、旋转、翻转规律

2.1 平移

平移规律主要考虑方向和步数两个维度。

  • 方向：上下左右；顺时针、逆时针；对角线
  • 步数：等差数列(不变也是公差为0的常数列)

平移规律的一些常见变形情况：

  • 内外圈分别平移
    • 方法：一般16宫格图形 + 主动画出内外圈 + 内外圈分别观察
  • 黑色块可能重叠
    • 方法：基本组成相同 + 某个图形少1个黑色块
  • 黑色块分不清楚
    • 方法：一个图形多个黑色块无法区分清楚，通过标号 + 找最简线路处理

2.2 旋转

旋转规律主要考虑方向和度数两个维度

  • 方向：顺时针、逆时针
  • 度数：等差数列，常考 30、45、60、90、120、150、180度等

2.3 翻转

翻转指沿翻转轴成轴对称，主要考察横轴翻转和竖轴反转

  • 横轴翻转：左右不变，上下翻

  • 竖轴翻转：上下不变，左右翻

3、静态位置

静态位置主要指：位置状态(位置关系)是稳定的。不一定组成相同，但一定存在明显的位置关系

主要包含上下左右、前后内外、相邻间隔、相离相交相切、平行垂直等

4、功能信息

功能信息主要指：一些提示性的相关信息

主要特征为：提示性信息都很明显、很显眼

功能点主要包括：黑点、球、圈等明显元素，主要可以分为以下常见类别：

  • 点元素本身位置：
    • 静态位置：上下左右、面内面外
    • 特殊位置：点上、线上(线的曲直、最长线/最短线)、角上(钝角/直角/锐角、最大角/最小角)
  • 点元素相对位置：
    • 多个点元素(常以两个点为主)
  • 功能箭头
    • 箭头指向：指向线(线的曲直、最长线/最短线)、指向角(钝角/直角/锐角、最大角/最小角)
    • 箭头位置：相对位置(平行/垂直)

三、样式规律

1、题型识别

当图形组成相似，优先考虑样式规律

该题目一般都为线段类别

2、遍历

特点

一般都分层观察(比如：九宫格的一行/两组型题型中的一组)

  • 层与层的组成元素完全相同
  • 层与层之间找元素全集( 缺啥补啥 )

3、加法

直接叠加

一个图叠加到另一个图上，需注意：

  • 叠加的方向性：木 和 口 叠加，根据方向不同，可以得到 杏、呆、困 等字
  • 遮挡性：参考位置规律中前后遮挡的第一组图
规则运算(黑白叠加)⚠️

有部分题型还涉及到了规则运算，例如(黑白叠加)，该题型考频较高

  • 图形组成相似：轮廓 + 分割区域相同 + 内部颜色/元素 不同
  • 该类题型需在题目中总结规律，在题目中应用规律，并结合选项进行排除

在这类题目中，需注意⚠️：

加法交换律不一定成立。即黑块 + 白块 不一定等于 白块 + 黑块

4、求同

去异求同

去异求同一般针对线条操作，留下线条相同的结果

样式求同

样式求同一般只针对图形

样式求同只关注图形本身是否相同(比如是否图形中都包含三角形)

和图形大小、比例、面积等都无关

5、求异

去同求异 ⚠️

去同求异一般针对线条操作，留下线条不同的结果，该题型考频较高

主要题型：主要题型为 九宫格/ 两组型题型

  • 3个图形有明显的相同线条
  • 线条之间加加减减、拼拼凑凑能有一些关系

该类题型还经常与位置规律相结合，比如求异之后，结果旋转九十度再得到最终结果。
